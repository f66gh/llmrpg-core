# 📅 日常固定触发事件表 (Daily Event Triggers) v0.4

> **核心机制**：通过多维判定（堕落值/治安度/时间/地点/NPC/道具）来决定事件走向。 **数值约束**：严格限定于 `金钱`、`堕落值`、`欲望值`、`治安度`、`开发度`。 **数值调整**：**[极慢速版]** 大幅降低数值收益，堕落是一个漫长的侵蚀过程。
