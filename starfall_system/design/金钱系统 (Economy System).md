# 💰 金钱系统 (Economy System) v1.0

> **核心机制**：极简生存压力。每天醒来，你都欠这座城市一笔钱。 **货币单位**：星币 (StarCoin / SC) **初始资金**：2000 SC (约两周生存资金) **时间单位**：时段 (上午/下午/傍晚/半夜)

## Ⅰ. 日常开销与惩罚 (Daily Expenses)

> **机制**：每天**凌晨 00:00** (半夜结束时) 自动结算。

### 1. 固定支出 (Fixed Costs)

- **账单总额**：**`-180 SC / 天`**
  - *包含*：学费分期、公寓租金、水电网费、基本餐费。
  - *描述*：这是在这个城市维持“体面女大学生”身份的最低成本。

### 2. 贫穷惩罚 (Debt Penalty)

- **余额 < 500 SC (拮据)**：

  - 焦虑

    ：琪琪会频繁查看手机余额，无法购买任何

    B级

    以上的服装或道具。

  - 省钱

    ：自动拒绝所有涉及消费的社交邀请（如喝咖啡、聚餐），导致错失部分剧情。

- **余额 < 0 SC (负债)**：

  - 停摆

    ：公寓断水断电，手机欠费停机（无法接任务）。

  - 强制打工

    ：第二天**上午**和**下午**会被强制锁定去干苦力（洗盘子/发传单），收益极低且无法自由行动。

  - 高利贷

    ：解锁“黑蛇”的高利贷选项。一旦借款，每天需偿还高额利息，否则会触发“肉偿”强制剧情。
