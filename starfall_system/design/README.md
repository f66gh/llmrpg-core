# 星陨：无垢之欲 Starfall: Innocent Lust

## **【故事梗概】**

**“所谓堕落，不过是理智选择了快乐。”**

在繁华与混乱共生的现代化都市——星陨市（Starfall City），21岁的**琪琪（Kiki）**是星陨大学公认的“高岭之花”。

拥有 B103(G)/W60/H92 的魔鬼身材，却总是用宽大的卫衣遮掩；拥有年级第一的超高智商，却总是挂着一副人畜无害的甜美笑容。

在所有人眼中，她是完美的优等生，是不可亵渎的纯白象征。

然而，在这副**【A级：纯白防线】的完美面具下，隐藏着一个扭曲的灵魂。

琪琪是一名“纯白堕落者”——她对两性知识一无所知，是一张彻头彻尾的白纸；但她的内心深处，却燃烧着一种名为“无垢之欲”**的疯狂渴望。她好奇失控的感觉，好奇被玷污的快感，好奇理智崩断那一刻的风景。

## **【双面生活】**

某天，琪琪意外被卷入了名为**“异魔（The Desire）”的超自然灾害中，并与神秘组织签订契约，成为了狩猎欲望的魔法少女**。

但这并不是一个正义战胜邪恶的童话。

- **白昼（Day）**：她是精明的伪装大师。为了支付每天高达 **180 SC** 的高昂生活成本，她周旋于家教学生、好色教授和健身教练之间，利用自己的美貌和智慧，在不越过底线的前提下榨取最大的利益。
- **黑夜（Night）**：她是危险的狩猎者。在法律失效的**C区**和诡异的**S区**，她穿着逐渐大胆的战衣，面对触手、粘液与精神控制。对她而言，这不仅是战斗，更是一场以身体为筹码的**“人体实验”**。

## **【核心冲突】**

随着**【堕落值】与【开发度】的攀升，琪琪发现自己正在发生不可逆转的变化：

原本敏感抗拒的A级身体**，开始适应甚至渴望异物的入侵；

原本只为了生存的金钱交易，逐渐变成了满足私欲的借口；

原本用来防身的魔法力量，副作用却让她的**【欲望值】**时刻处于过载边缘。

是坚守最后一点理智，利用规则成为掌控一切的恶堕女王？

还是彻底放纵本能，沦为这座城市里人人可欺的公共肉便器？

**在这个秩序崩坏的城市里，请小心。猎人与猎物的身份，只隔着一件薄薄的蕾丝内衣。**
