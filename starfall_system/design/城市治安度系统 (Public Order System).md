# ⚖️ 城市治安度系统 (Public Order System)

> **核心机制**：反映星陨市（Starfall City）当前的社会秩序与道德底线。 **数值范围**：**100 ~ -100** (共六个级别) **自然衰减**：每日凌晨 00:00，治安度自动 **-2** (熵增定律，秩序总是趋于混乱)。

## Ⅰ. 治安等级详解 (Order Levels)

### 【Lv.5：乌托邦】 (Utopia)

- **数值区间**：**100 - 81**
- **城市风貌**：
  - 街道一尘不染，阳光明媚，警力充足且友善。
  - 监控覆盖率 100%，没有任何视觉死角。
  - 充满欢声笑语，空气中弥漫着安全感。
- **NPC 行为**：
  - **路人**：彬彬有礼，目不斜视。即使看到穿着性感的琪琪，也只会暗中欣赏或礼貌赞美。
  - **异魔**：几乎绝迹，只能躲在最阴暗的下水道里瑟瑟发抖。
  - **特殊事件**：几乎为零。琪琪可以毫无顾虑地在半夜穿超短裙散步。

### 【Lv.4：模范都市】 (Model City)

- **数值区间**：**80 - 61**
- **城市风貌**：
  - 繁华有序，虽然偶尔有小摩擦，但很快会被解决。
  - 夜晚的霓虹灯代表着繁荣而非暧昧。
- **NPC 行为**：
  - **路人**：偶尔会有大胆的目光，但绝不敢上手。搭讪也是基于礼貌的。
  - **异魔**：只敢在深夜的 C 区活动。
  - **琪琪体验**：这是琪琪最舒适的区间，既能享受“被注视”的虚荣感，又不用担心人身安全。

### 【Lv.3：浮华表象】 (Gilded Age)

- **数值区间**：**60 - 41**
- **城市风貌**：
  - 表面光鲜，背地里暗流涌动。涂鸦开始出现在小巷，监控摄像头偶尔会“故障”。
  - 红灯区的边界开始模糊，向商业区渗透。
- **NPC 行为**：
  - **路人**：眼神开始变得粘稠。地铁上的咸猪手事件频发，但只要瞪回去，对方还会收敛。
  - **异魔**：开始伪装成人类混入 B 区，寻找落单的猎物。
  - **琪琪体验**：必须时刻保持警惕（或腹黑模式），利用小手段惩罚越界者。

### 【Lv.2：混乱前夜】 (Pre-Chaos)

- **数值区间**：**40 - 21**
- **城市风貌**：
  - 垃圾随处可见，路灯经常损坏。警车警笛声成为背景音。
  - “寻人启事”贴满了电线杆。
- **NPC 行为**：
  - **路人**：充满了戾气和色欲。路边的混混会公然对琪琪吹口哨、评头论足，甚至试图阻拦去路。
  - **治安局**：开始推诿责任，对轻微的性骚扰视而不见。
  - **琪琪体验**：穿着稍微暴露一点（C级以上）出门，就会成为移动的靶子。如果不反击，就会被进一步侵犯。

### 【Lv.1：罪恶之城】 (Sin City)

- **数值区间**：**20 - 0**
- **城市风貌**：
  - 秩序崩坏。商店早早关门，窗户钉上木板。
  - 街道上弥漫着某种甜腻而腐烂的气味（欲望具象化）。
- **NPC 行为**：
  - **路人**：已经不能称之为“人”。他们的眼中只有原始的欲望，随时准备扑向猎物。公共场合的性行为不再被制止。
  - **异魔**：堂而皇之地在白天出现，甚至占据了某些建筑物。
  - **琪琪体验**：只要出门就是战斗。每一寸肌肤的裸露都是在邀请侵犯。

### 【Lv.0：人间炼狱】 (Hell on Earth)

- **数值区间**：**0 - -100**
- **城市风貌**：
  - **里世界完全降临**。天空变成紫色，建筑物扭曲成肉块状。
  - 法律彻底失效，唯一的规则是“强权”与“交配”。
- **NPC 行为**：
  - **全员恶堕**。即使是原本正义的警察或老师，也变成了只会追逐快感的行尸走肉。
  - **琪琪体验**：在这个级别下，琪琪如果不具备强大的力量（或彻底堕落顺从），将寸步难行。这里是重口味玩法的温床。

## Ⅱ. 琪琪的行为影响 (Influence)

> **机制**：琪琪作为拥有魔力的核心人物，她的选择直接干涉着城市的命运。

### 📈 正面影响 (增加治安度)

- **打击犯罪**：
  - 击败游荡的混混/痴汉：`+1`
  - 捣毁异魔巢穴：`+5`
  - 击败区域 Boss：`+15`
- **公益活动**：
  - 兼职图书馆管理员（整理秩序）：`+1`
  - 兼职交通协管员（制止违章）：`+1`
  - 在医院做义工（治愈人心）：`+2`
- **正向互动**：
  - 帮助迷路的小孩/老人：`+1`
  - 拒绝并惩罚不当搭讪：`+1` (杀鸡儆猴)
  - 穿着得体（A/B级）在公共场合展现优雅形象：`+0.5` (净化眼球)

### 📉 负面影响 (降低治安度)

- **战斗失败/妥协**：
  - 战斗战败（被异魔侵犯）：`-3` (魔力被污染，助长邪恶)
  - 面对骚扰选择忍气吞声：`-1` (纵容风气)
- **堕落行为**：
  - 穿着暴露（D/E级）在公共场合招摇过市：`-2` (引发路人邪念)
  - 在公共场合发生性行为/露出：`-5` (极其恶劣的示范)
  - 拍摄并传播私密照片（流出）：`-3`
- **助纣为虐**：
  - 被恶堕后协助异魔捕猎：`-10`
  - 在打工场所（如酒吧）提供特殊服务：`-2`

## Ⅲ. 治安度对琪琪的反向影响 (Feedback)

- **如果治安度高 (Lv.4-5)**：
  - **好处**：能够安全地进行日常活动，提升属性。
  - **坏处**：想要寻找刺激（增加欲望值/堕落值）变得困难，需要主动去 C 区或 S 区寻找机会。
  - **特殊**：如果琪琪在这个环境下穿着过于暴露，会受到来自 NPC 的**道德谴责**（San 值下降）。
- **如果治安度低 (Lv.1-2)**：
  - **好处**：到处都是“机会”，刷堕落值和欲望值非常容易。
  - **坏处**：日常活动受阻（比如便利店关门买不到补给）。战斗频率极高，体力消耗大。
  - **特殊**：如果琪琪在这个环境下依然保持清纯（A级装），会成为**稀有的独角兽**，引发更疯狂的追逐（Boss 级敌人出现率增加）。
